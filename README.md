📌 Project Overview
This project provides a comprehensive analysis of global tourism trends, focusing on traveler spending behavior, regional revenue distribution, and profit margins. By visualizing key performance indicators (KPIs), this dashboard helps stakeholders identify high-value tourism sectors and optimize regional marketing strategies.

📊 Key Features & Insights
Spending vs. Volume Analysis: A scatter plot analyzing the correlation between the number of tourists and their average spend to identify "luxury" vs. "mass-market" traveler segments.

Regional Performance: Revenue breakdown by East, North, South, and West regions, highlighting that the North region currently leads in total revenue generation.

Popularity by Tour Type: A distribution analysis showing that Wildlife and Religious tourism are the top-performing categories by volume.

Profitability Metrics: A frequency distribution of profit bins to visualize how often trips achieve high-margin status.
